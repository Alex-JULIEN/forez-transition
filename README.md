# Forez Transition & Climat — site Jekyll minimal

Déployable sur GitHub Pages.  
Contient un design *low-tech* : peu de ressources, pas de JS, CSS minimal.

Instructions rapides :
1. Push sur un repo GitHub.
2. Activer GitHub Pages (branche `main`, dossier `/`).
3. Dans Settings > Pages, indiquer le domaine personnalisé `forez-transition.fr`.
4. Configurer le DNS chez OVH (voir README ou docs GitHub Pages).
